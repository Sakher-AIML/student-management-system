Smart Student Management System

A beginner-friendly full-stack web application for managing students, subjects, attendance, marks, and reports. Built using HTML, CSS, JavaScript, PHP, and MySQL.

Read Full Documentation Open PHP App (local only)
Important: GitHub Pages only hosts static files. This page is a static project landing page. The full system requires PHP + MySQL, so run it locally using XAMPP and open http://localhost/SSMS/login.php.

Core Modules

Admin Login

Secure login with session-based access control for admin operations.

Student Management

Add, view, search, edit, and delete student records.

Attendance Management

Record attendance by student, subject, date, and status.

Marks and Reports

Save marks, auto-calculate grades, and print student reports.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP (MySQLi)
  • Database: MySQL
  • Local Server: XAMPP (Apache + MySQL)

Quick Start (Local)

Install XAMPP and start Apache + MySQL.
Copy this project folder to xampp/htdocs/SSMS.
Import database.sql via phpMyAdmin.
Open http://localhost/SSMS/login.php and login using admin / admin123.
